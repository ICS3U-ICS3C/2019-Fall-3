Module 1 (Sept 16 - 27)

The main focus of this module will be using HTML and CSS to create a personal webpage to post your coding projects, hosted on GitHub. It can also function as a preofessional portfolio. Note: Any material presented in this module is fair game for the quiz.

Styling with CSS

Next steps for continuing to build your website

  1. Complete Section 1 (two lessons), Section 2 (two lessons), and Section 3 (one lesson) of the "Introduction to CSS" course on Codecademy
  2. If you'd like to review the basics of what you've learned so far about HTML and CSS, Mozilla Web Docs offers summary notes on basic HTML and CSS
  3. Your GH site is where you'll host your JavaScript code projects later. Make sure it meets the following minimum requirements:
    • A section about you (your tech skills, interests, experiences, any personal projects coding/tech projects you're working on.
    • A section/page to post your assignments. You may want to use buttons as placeholders, or images (placeholder.it)
    • Clear headings, subheadings, images, links, videos, where appropriate
    • An external CSS file. It should be linked to every page in your site
    • All pages should be linked and all sections of your site easily reachable from all other parts
    • The content of your site should be engaging, informative, and aesthtically pleasing
    • Remember: If you use elements of someone else's code, credit the source within a comment in your code
