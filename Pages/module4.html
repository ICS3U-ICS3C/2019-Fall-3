Module 4 (Nov 4 - Nov 15)

In this section, we'll learn more about Javascript string manipulation and introduce counting and encoding. Note: Any material presented in this module is fair game for the quiz.

JS Skills: String Manipulation

Strings

  • You should know: How to manipulate strings using various methods in JS; How to determine properties of a string, such as .length; How to manipulate a string the way you would an array
  • Summary of some useful string methods and properties to know
  • Try this quiz to test your understanding of String Methods


JS Strings Exercises

  1. Create a function that takes one string argument and returns the length of the string

  2. Create a function that accepts two string arguments (userName, password), converts the username to all lowercase, and checks to see if they match a username and password that you set. The function should return "True" if it's a match, and "False" if it's not.

  3. Create a function that takes one string argument, msg, and returns the string with eVeRy OtHeR lEtTeR cApItAlIzEd.
