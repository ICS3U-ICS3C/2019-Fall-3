Module 0 (Sept 4 - 13)

We'll begin by getting to know our way around GitHub, a tool for keeping our files organised and for submitting assignments

Getting Started

You may be required to create a number of new online accounts to access course content and submit assignments. Take care to keep track of your various accounts to avoid wasting time on forgotten logins and passwords. You may want to bookmark this class site and other frequently accessed sites for the course.

  1. Review the Syllabus and the Statement on Plagiarism here
  2. Read the Course Outline
  3. Read the ACM's Programming Code of Ethics
  4. Create a Codecademy account
  5. Create a Github account

File Maintenance and Version Control

We'll be using GitHub (link at top left) to organise our files. It lets you save different versions of a project as you go along, and keeps shared work organised when working on group projects. It's also where you'll be accessing and submitting coding assignments, but we'll get to coding later.

  1. Say Hello to GitHub! It's a steep learning curve, so expect that it will take us some time to figure it out.
  2. Explore GitHub and complete Asst-0. NOTE: Click only on the Assignment link corresponding to the section of the course you are registered in, A or B

Finished early? You can work ahead on any of the following:

  • Make your own GitHub webpage (like this one) by creating a gh-pages branch from your own Repository. We will use your github page to organise your JavaScript assignments later in the course. GitHub hosts your pages for free!
  • If you want to get fancy with your GitHub READMEs, and generally be more stylish, look at Markdown Syntax, used to style .md files
  • Begin working through the "Introduction to HTML" course on Codecademy
