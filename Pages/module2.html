Module 2 (Sept 30 - Oct 11)

We begin this module with and introduction to basic programming in JavaScript. By the end of this module, you will be able to create an external JS file and link it to your HTML pages; you will be able to create an HTML button that, when clicked, calls a JS function in your JS file; you will be able to prompt the user for input and display output in an alert box or log to the console; you will be able to us conditional statements and logical operators to solve coding problems with JavaScript. Any material presented in this module is fair game for the quiz.

JS Skills I

Complete the Introductory JavaScript Tutorials below:

  1. Internal vs. External Scripts: W3Schools Tutorial
  2. JS Variables: W3Schools Tutorial
  3. Output: Alerts, Logging to the Console: W3Schools Tutorail
  4. Conditional (if) Statments: W3Schools Tutorial
  5. Comparisons and Logical Operators: W3Schools Tutorial
  6. Functions: creating functions, calling functions, parameters and arguments, functions with multiple/single/no arguments. See the W3Schools Tutorial or try this quiz to test your understanding of Functions
