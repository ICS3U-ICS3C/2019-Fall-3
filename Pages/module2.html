JS Skills I

By now you should know how to create an external JS file and link it to your HTML pages (1). You should be able to create an HTML button that, when clicked, calls a JS function in your JS file (2) You should know how to prompt the user for input and display output in an alert box or log to the console (3) Complete the tutorials below to review these topics and to learn about conditional statements (4) and logical operators (5) in JS.

  1. Internal vs. External Scripts: W3Schools Tutorial
  2. JS Variables: W3Schools Tutorial
  3. Output: Alerts, Logging to the Console: W3Schools Tutorail
  4. Conditional (if) Statments: W3Schools Tutorial
  5. Comparisons and Logical Operators: W3Schools Tutorial
  6. Functions: creating functions, calling functions, parameters and arguments, functions with multiple/single/no arguments. See the W3Schools Tutorial or try this quiz to test your understanding of Functions
  7. Arithmetic Operators: W3Schools Tutorial

JS Practice Exercises

  1. Write a JavaScript program that prompts the user for two integers and logs the larger of the two to the console.
    Sample input: 3, 18
    Expected output: 18
    2.  
function biggerNum(){

var num1=parseInt(prompt("Enter first number"));
var num2=parseInt(prompt("Enter second number"));

	if (num1>num2){
		console.log(num1+" is the bigger number");
	}else if(num2>num1){
		console.log(num2+" is the bigger number");
	}else if(num1==num2){
		console.log("The numbers are the same");

		console.log("You didn't enter two numbers");

  2. Write a JavaScript function to determine the product of three integers and use a conditional statement to state whether the product is positive or negative. Display a message in an alert box indicating the sign.
    Sample input: 3, -7, 2
    Expected output: The product is a negative number

  3. Write a JavaScript funtion that takes three integer inputs and lists them in order from largest to smallest. Display the result in an alert box.
    Sample input: 0, -1, 4
    Expected output: 4, 0, -1
